## Noun System

| Pattern | What it means | Verse that triggered it |
|---------|--------------|------------------------|
| प्रथमा = nominative | The -ः/-ाः ending = subject of sentence | 1.1 (धृतराष्ट्रः, मामकाः) |
| सप्तमी = locative | The -े ending *means* "in/at" — no separate preposition needed | Lesson 2 (वृन्दावने) |
| द्वितीया = accusative | The -म्/-ान् ending = object being acted on | 2.4 (भीष्मम्, द्रोणम्) |
| तृतीया = instrumental | The -भिः ending = "by/with" | 2.4 (इषुभिः — with arrows) |
| सम्बोधन = vocative | Shortened form (-अ endings) — addresses someone, never in the action | 2.4 (मधुसूदन, अरिसूदन) |
| विशेषण-विशेष्य agreement | Adjective must match noun in लिंग + वचन + विभक्ति (सामानाधिकरण्य) | 2.5 (महानुभावान् गुरून्) |
| Adjective identification | Endings narrow candidates; meaning picks the final match | 1.1 (समवेताः ↔ युयुत्सवः) |

## Verb System

| Pattern | What it means | Verse that triggered it |
|---------|--------------|------------------------|
| Finite verb = sentence anchor | Only finite verb is conjugated for पुरुष + वचन + लकार; everything else orbits it | 2.5 (भुञ्जीय) |
| लङ् लकार identification | अ- prefix + -त् suffix = past tense | 1.1 (अकुर्वत) |
| लृट् लकार identification | -ष्य-/-स्य- infix = future tense | 2.4 (प्रतियोत्स्यामि) |
| लोट् लकार identification | Imperative mood — Krishna's instructions | 2.3 (उत्तिष्ठ, गमः) |
| विधिलिङ् identification | -ीय/-यात् endings = "should/ought to" — the mood of rhetorical anguish | 2.5 (भुञ्जीय) |
| मा स्म + लोट् | "Do not [verb]" — negated imperative | 2.3 (मा स्म गमः) |
| प्रथम पुरुष flip | Sanskrit प्रथम = English 3rd person (he/she/it). Counted from outside in. | 1.1 (अकुर्वत — they did) |

## Kṛdanta (Verb-Derived Non-Finite Forms)

| Pattern | What it means | Verse that triggered it |
|---------|--------------|------------------------|
| Past passive participle (कृदन्त) | Verb-derived adjective — agrees with a noun, does NOT anchor the sentence | 1.1 (समवेताः from सम् + अव + √इ) |
| Absolutive (हत्वा form) | "Having [done]" — describes action before the main verb | 2.5 (हत्वा, अहत्वा) |
| Negative absolutive (अहत्वा) | अ + absolutive = "without [doing]" | 2.5 (अहत्वा — without killing) |
| Infinitive (भोक्तुम् form) | "To [do]" — states purpose, cannot anchor sentence | 2.5 (भोक्तुम् — to eat) |

## Decode Method

| Pattern | What it means | Where learned |
|---------|--------------|---------------|
| पदच्छेद → अन्वय → हिंदी → English | The full decode sequence | Conversation |
| अन्वय follows SOV | Reorder into Subject → Object → Verb (not SVO) | Conversation |
| Hindi before English | Stays in SOV mode, doesn't force English SVO on Sanskrit structure | Conversation |
| Giveaway test | Remove word X — does the sentence still work? If yes, X is not the finite verb | 2.5 (removing भोक्तुम् vs removing भुञ्जीय) |
| Endings narrow, meaning picks | विभक्ति matching gives candidates; context selects the right pairing | 1.1 (समवेताः — which noun does it describe?) |
| Sandhi comes last | Understand the expected form first; then the change makes sense | bvsiitm explainer |
